## 1. 기획 배경

### 산업 현장의 현실적인 문제

반도체 제조 공정에는 다음과 같은 문제가 반복적으로 발생합니다.

- **결함 데이터의 라벨링 비용이 매우 큼**
- 신규 공정 / 신규 장비 도입 시  
  → **충분한 학습 데이터 확보가 어려움**
- 검사 기준이 사람·시점에 따라 달라질 수 있음
- 실제 현장에서는  
  **“완벽한 모델”보다 “일관된 판단 시스템”이 더 중요**

해당 프로젝트는 이러한 현실을 반영하여,  
**소량의 무라벨 데이터만 주어진 상황에서 어떻게 신뢰 가능한 판단을 자동화할 것인가**를 핵심 문제로 삼습니다.

## 2. 문제 정의

### 주어진 조건
- 라벨이 없는 **예시 이미지 20장**
- 실제 평가 대상은 **100장 이상의 이미지**
- 이미지에는 트랜지스터 구조가 포함되어 있으며,
  - 결함이 존재하거나
  - 정상 상태일 수 있음

### 해결해야 할 문제

> **무라벨 환경에서 이미지 기반으로 결함 유무를 판단하고,  
판단 결과를 스스로 검증·보정하며,  
최종 결과를 자동으로 제출하는 AI Agent를 설계하라.**

## 3. 접근 전략 개요

본 프로젝트는 단일 모델의 성능 극대화보다,  
**에이전트 관점의 문제 해결 구조**를 설계하는 데 초점을 둡니다.

### 핵심 아이디어

- 한 번의 판단으로 끝내지 않는다
- 판단 과정 자체를 **검증하고 보정**한다
- 결과가 이상할 경우 **정책적으로 재확인**한다
- 전체 과정은 **완전 자동화**한다
